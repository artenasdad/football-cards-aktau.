Football Cards Aktau

Как использовать:
1. Поместите файл 'index.html' и 'style.css' в корень сайта.
2. В папке 'images' добавьте ваше изображение карты под именем 'ronaldo.png'.
3. Загрузите файлы на GitHub Pages или любой другой статический хостинг.
4. Готово! Ваш сайт будет доступен по ссылке GitHub Pages.

Чтобы добавить новые карточки, скопируйте блок `<div class="card">...</div>` в index.html и замените данные.
